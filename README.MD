# Modern Developer Portfolio ✨

A sleek, responsive, single-page portfolio website designed with a dark cyberpunk/developer aesthetic. It features smooth interactive animations, an integrated CSS timeline, and an immersive custom 3D particle field background.

🔗 **Live Link:** [https://yourusername.vercel.app](https://yourusername.vercel.app)

---

## 🚀 Features

*   **3D Interactive Background:** Built using **Three.js**, featuring a dynamic particle field and floating torus-knot geometric structure that reacts seamlessly to mouse movements.
*   **Modern Interactive UI:** Glassmorphism card layouts with responsive 3D hover effects (`transform-style: preserve-3d`) that flip to reveal project features on click.
*   **Clean Visual Hierarchy:** Uses typography pairings of **Space Grotesk** for clean layout scanning and **JetBrains Mono** for technical code accents.
*   **Fully Responsive:** Designed mobile-first using clean CSS Grid and Flexbox structures to adjust layout fluidly across viewports.
*   **Scroll-Driven Animations:** Utilizes the high-performance native JavaScript `IntersectionObserver` API to animate elements into view smoothly as the user scrolls.

---

## 🛠️ Technical Stack

*   **Frontend:** HTML5, CSS3 (Custom Variables, CSS Grid, Flexbox), JavaScript (ES6+)
*   **3D Graphics:** Three.js (WebGL Renderer)
*   **Fonts & Icons:** Google Fonts (Space Grotesk, JetBrains Mono)
*   **Deployment & Hosting:** Vercel
